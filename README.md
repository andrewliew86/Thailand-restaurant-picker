# Ari Eats - Thailand Restaurant Picker

A modern web application built with React and Vite to help you discover and pick restaurants in Thailand. Features a clean, responsive UI designed with Tailwind CSS and Lucide icons.

## Features

- Browse a curated list of Thai restaurants
- Filter restaurants by cuisine, location, or price range
- Random restaurant picker for when you're indecisive
- Responsive design for mobile and desktop

## Tech Stack

- **Frontend**: React 19
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Linting**: ESLint

## Setup and Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/thailand-restaurant-picker-app.git
   cd thailand-restaurant-picker-app
   ```

2. Navigate to the app directory:
   ```bash
   cd ari-eats
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173` to view the app.

## Available Scripts

- `npm run dev` - Start the development server
- `npm run build` - Build the app for production
- `npm run preview` - Preview the production build locally
- `npm run lint` - Run ESLint for code linting
